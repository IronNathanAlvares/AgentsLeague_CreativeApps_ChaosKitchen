## 🤖 AI API CONFIGURATION - Powering The Recipe Generator

</div>

Chaos Kitchen uses **AI-powered recipe generation** to create wild, creative dish cards from your ingredient combinations. The backend supports any **OpenAI-compatible API**.

### 🎯 **What AI API Was Used?**

This project was built using **Ollama with Mistral** - a **100% free, local, open-source** AI solution that runs entirely on your machine with zero API costs and complete privacy!

### 🔧 **Configuration Setup**

1. **Copy the environment template:**
   ```bash
   cp .env.example .env
   ```

2. **Edit `.env` with your API credentials:**
   ```env
   # Ollama Mistral (as used in this project)
   MODEL_API_BASE_URL=http://localhost:11434/v1
   MODEL_NAME=mistral
   # No API key needed for local Ollama!
   ```

## 💬 FAQ - Chef's Answers

**❓ Why isn't my progress saving?**  
→ Check browser storage settings. Enable cookies/localStorage.  
→ Private/Incognito mode? Progress won't persist (by design).  
→ Try a different browser. Firefox/Chrome/Edge work best.

**❓ Which mini-game gives the MOST stars?**  
→ **Accuracy Shooter** (50-100⭐) is the jackpot!  
→ But **Memory Match** has the best multiplier (0.25x).  
→ Try all 4 to find YOUR best game!

**❓ How long to unlock EVERYTHING in the shop?**  
→ 5-10 hours of regular play (averaging 60-80⭐/round).  
→ Speed demons can do it in 4 hours with perfect scores.  
→ Casuals will get there eventually. Enjoy the chaos!

**❓ Audio not working?**  
→ Browsers require user interaction first. Click the audio toggle.  
→ Check browser permissions (allow sound for localhost).  
→ Some browsers block Web Audio API by default.

**❓ Can I export/share my recipes?**  
→ Currently localStorage only (no cloud sync... yet?).  
→ Screenshot your favorite dish cards!  
→ Or manually copy the text. Old school!

**❓ Will there be multiplayer?**  
→ Not in this version, but... never say never! 👀

## 🛠️ TROUBLESHOOTING - When Chaos Goes Wrong

### 🚨 **Problem: npm install fails**
```bash
# Solution 1: Nuclear option
rm -rf node_modules package-lock.json
npm install

# Solution 2: Check Node version
node --version  # Must be 18+

# Solution 3: Clear npm cache
npm cache clean --force
```

### 🚨 **Problem: Dev server won't start**
```bash
# Port 5173 already in use?
npm run dev -- --port 3000

# Or kill the zombie process
# Windows: taskkill /F /IM node.exe
# Mac/Linux: killall node
```

### 🚨 **Problem: Build errors**
```bash
# Clear TypeScript cache
rm -rf node_modules/.vite
npm run build

# Still failing? Restart everything
# Close VS Code, reopen, try again
```

### 🚨 **Problem: Game assets not loading**
```bash
# Hard refresh browser
# Windows: Ctrl + Shift + R
# Mac: Cmd + Shift + R

# Clear browser cache completely
# Or try incognito mode
```
